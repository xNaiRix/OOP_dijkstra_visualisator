# Визуализатор алгоритма Дейкстры

Проект на C++/Qt для интерактивной визуализации алгоритма Дейкстры на графах.

## Возможности
- Создание графа в графическом редакторе (вершины и рёбра с весами)
- Выбор стартовой вершины
- Пошаговая визуализация алгоритма Дейкстры
- Автоматическое выполнение с анимацией
- Отображение расстояний до вершин
- Подсветка текущей вершины и рассматриваемых рёбер

## Требования
- Qt 6 (Core, Widgets)
- CMake 3.16+
- Компилятор с поддержкой C++17 (GCC, Clang, MSVC)
  - Для Windows рекомендуется MinGW (совместимый с Qt для MinGW)

## Сборка
```bash
mkdir build
cd build
cmake ..
cmake --build .
```
Или откройте проект в Qt Creator.


## Устранение ошибок CMake
Если возникает ошибка **"Could not find a package configuration file provided by Qt6"**:
- Проверьте, что путь в `-DCMAKE_PREFIX_PATH` ведёт к каталогу, содержащему `lib/cmake/Qt6`.
- Убедитесь, что установлен Qt6 с компонентами Core и Widgets.
- Можно также установить переменную окружения `Qt6_DIR` (например, `set Qt6_DIR=D:/Qt/6.9.2/mingw_64/lib/cmake/Qt6`).


## Использование
1. Запустите программу.
2. Нажмите "Добавить вершину" и щёлкните на холсте для создания вершин.
3. Нажмите "Добавить ребро", затем выберите две вершины для соединения (вес запрашивается).
4. Нажмите "Установить старт" и выберите стартовую вершину.
5. Нажмите "Шаг" для пошагового выполнения алгоритма или "Запуск" для автоматической анимации.
6. Используйте "Сброс" для очистки визуализации.

## Интерфейс
- **Холст графа**: центральная область для рисования графа.
- **Панель инструментов**: кнопки управления.
- **Статусная строка**: информация о текущем режиме и состоянии алгоритма.

## Управление и шорткаты
- **Правая кнопка мыши (ПКМ)** на пустом месте → создание вершины.
- **Левый клик** по вершине → выделение вершины.
- **Shift + левый клик** по двум вершинам → создание ребра между ними (запрос веса).
- **Delete** → удаление выделенной вершины или ребра.
- **Двойной щелчок** по ребру → изменение его веса.
- **Контекстное меню** (ПКМ на вершине/ребре) → дополнительные действия (удалить, изменить вес, сделать стартовой).

## Цветовая схема
- **Серая вершина**: обычная
- **Зелёная вершина**: стартовая
- **Синяя вершина**: текущая (обрабатываемая)
- **Светло-голубая вершина**: посещённая
- **Чёрное ребро**: обычное
- **Оранжевое ребро**: рассматриваемое на текущем шаге
- **Красное ребро**: входит в кратчайший путь

## Алгоритм
Реализован классический алгоритм Дейкстры для неориентированных взвешенных графов с положительными весами. Поддерживается пошаговое выполнение с визуализацией каждого этапа.
